# PLANET B
![Image of a galaxy and planet](https://github.com/jjwallace/planetb/blob/main/notes/pictures/galaxy.jpg)

## Install

```bash
git clone <this repository>
cd planetb
npm install

```

### Client
```bash
cd planetb
npm install

```

### Server
```bash
cd server
npm install

```

## Running

### Client
```bash
npm start
```

### Server
```bash
cd server
npm start
```

## Troubleshooting

User node version 14
```
nvm use 14
```
